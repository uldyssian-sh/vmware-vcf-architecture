# Installation Guide

## System Requirements

- Python 3.8+
- Docker 20.10+
- Git 2.25+
- 2GB RAM minimum (4GB+ recommended)

## Quick Installation

```bash
curl -fsSL https://raw.githubusercontent.com/uldyssian-sh/vmware-vcf-architecture/main/scripts/setup.sh | bash
```

## Manual Installation

1. Clone repository
2. Run setup script
3. Configure environment
4. Start application
